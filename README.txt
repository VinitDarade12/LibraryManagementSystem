# Library Management System

A full-stack web application for managing books and users in a library.

## Structure
- `/backend` — Spring Boot Java REST API + SQL
- `/frontend` — React.js user interface.

## Setup Instructions

1. Install backend: configure `application.properties`, run `mvn spring-boot:run`
2. Install frontend: `npm install && npm start` in `/frontend`
3. Access React app at `localhost:3000` and backend at `localhost:8080`.
